# Book Haven

Book Haven is a Flask-based web application for book enthusiasts to manage their reading lists, review books, and engage in vibrant community discussions. Users can add books to their personal lists, share their thoughts through reviews, and join a general forum to discuss their reading experiences—all in one place.

## Features

- **User Authentication:**  
  Secure sign-up, login, and logout functionality using Flask-Login.

- **Book Management:**  
  Create and manage personalized reading lists with categories like "To Read," "Reading," and "Finished."

- **Review System:**  
  Write, rate, and view reviews for individual books.

- **Community Discussion Panel:**  
  Engage in a general forum where users can start discussion threads, reply to others, and filter conversations using tags. This feature fosters community interaction without cluttering the interface.

- **Tagging & Filtering:**  
  Add tags to discussion threads to quickly find topics related to specific books, authors, or genres.

- **Database Integration:**  
  Built on PostgreSQL and managed with SQLAlchemy, ensuring robust data management.

- **Responsive UI:**  
  Clean, responsive front-end built with HTML, CSS, JavaScript, and Jinja2 templates.

## Tech Stack

- **Backend:** Python, Flask, Flask-Login, Flask-SQLAlchemy, Flask-Migrate  
- **Database:** PostgreSQL  
- **Frontend:** HTML, CSS, JavaScript, Jinja2 Templates  
- **Version Control:** Git, GitHub
